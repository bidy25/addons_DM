ORANGE MONEY

Overview

This module is used to manage Orange Money payment from Odoo. It offers to the customer a way to pay direclty from Odoo with a control of payment
Orange Money is a Mobile Money service in Madagascar powered by Orange, for more details about Orange Money, click here

This module is added to the list of native payment mode in the module Invoicing, after installation.

Go to Invoicing / Configuration / Payment Acquirers.

Configuration

Go to Invoicing / Configuration / Payment Acquirers

  1. Click on Activé
  2. Set Client ID, Client Secret, Autorisation from Moobipay in the payment acquirer configuration
  3. Click on the button "Obtenir Header"

Usage

  • Step 1: Add product to cart

    Add to cart all the products that you want to buy

  • Step 2: Add customer address

    Fill the information about the customer

  • Step 3: Choose payment method

    Choose Orange Money as payment method
